
# Automated Design Deployment 프로젝트 진행 요약

## 1. 파일 구조 및 모듈 정리
- 루트 js 파일들을 목적별로 이동 및 정리, src/func, src/ui, src/infra 등으로 세분화
- 모든 내부 모듈을 CommonJS(require/module.exports)로 통일
- import/export 혼용 문제 해결 및 require 경로 일괄 수정

## 2. 서버/유틸 파일 구조화 및 실행 스크립트 추가
- file-server.js, kill-4000.js 등 src/infra, scripts 등으로 이동
- package.json에 "file-server", "kill-4000" 등 실행 스크립트 추가

## 3. 디렉토리 탐색기 UX/UI 개선
- 네비게이션, 상위 폴더 이동, 아이콘, 스타일, 마우스 포인터, 하이라이트, 스크롤, 폴더-파일 순서, 디자인 개선

## 4. 즐겨찾기 기능 개선
- 서버별 즐겨찾기, 추가/삭제, 버튼 디자인 개선, 서버 접속 시에만 활성화
- app-data.json에 통합 저장, 이름 지정, 삭제, 클릭 시 바로 이동

## 5. 파일 업로드/다운로드
- 첨부파일 base64 전송, 이미지/텍스트 구분, 업로드시 _sned 네이밍, 용량제한 해제
- 업로드 후 경로 새로고침, 파일명 중복시 덮어쓰기 확인, 업로드 후 로그 및 알림

## 6. 에디터/비교/미니맵
- 파일 선택 시 에디터 1개만 표시, 첨부파일은 전송 전 미리보기 없이 알림창
- 비교(diff) 기능, 동일한 줄은 표시하지 않음, 하이라이트, 스크롤, 미니맵, 가로 스크롤, 레이아웃 개선

## 7. 로그/콘솔/알림
- 로그 패널 하단 고정, 숨김/최소화, 로그 디자인 개선, 네트워크/에러/상태 로그 출력, 콘솔창 스크롤 분리

## 8. 네이티브 메뉴/시스템 메뉴
- Electron 네이티브 메뉴에 접속/끊기, File/Edit 등 메뉴 추가, 접속 시 UI 숨김/표시 전환

## 9. app-data.json 통합
- 서버/즐겨찾기 등 모든 데이터 하나의 json에 저장, 삭제/추가/수정 반영, server-list.json 코드 제거

## 10. 빌드/배포/설치형
- electron-builder, NSIS로 설치형/포터블 빌드, npm run dist:installer로 배포, 설치형 exe 제공, 실행/설치 분리

## 11. 버그 수정/기타
- import/export 혼용, require 오류, 경로/플랫폼 문제, base64 이미지 손상, 업로드 용량, 파일명, 경로, 로그, UI/UX 등 다수 버그 수정 및 개선
